[작품명] 강남대학교 도서관 올인원 네이티브 앱 (KNU Library App)
1. 제작 동기 (Motivation)
기존 학교 도서관 서비스는 여러 기능이 파편화되어 있거나 로딩 속도가 느려 학생들이 도서관을 이용할 때 불편함을 겪는 경우가 많았습니다. 특히 가장 빈번하게 사용하는 **'출입용 QR 코드'**를 화면에 띄우기 위해 여러 번의 터치를 거치거나 긴 대기 시간을 감수해야 했습니다.

이러한 불편함을 해소하고자, 도서관 출입, 열람실 좌석 예약 및 관리, 도서 대출 현황 조회 등 학생들에게 꼭 필요한 핵심 기능들을 스크롤 없이 한눈에 파악할 수 있는 단일 홈 화면(Single Home Screen) 에 통합하였습니다. 궁극적으로 앱 실행 즉시 지연 없이 도서관 서비스를 이용할 수 있는 최적의 사용자 경험(UX)을 제공하기 위해 본 프로젝트를 기획하게 되었습니다.

2. 작품 설명 (Project Description)
본 작품은 React Native와 Expo 프레임워크를 기반으로 자체 개발한 강남대학교 도서관 전용 모바일 애플리케이션입니다. 시스템의 안정성과 유지보수성을 극대화하기 위해 엄격한 계층형 클린 아키텍처(Strict Layered Architecture)를 도입하여 설계되었습니다.

[주요 기능 및 특징]

1. 단일 대시보드 홈 화면 (Single Dashboard)
- 출입 QR 코드, 이용 중인 좌석의 남은 시간, 대출 중인 도서 요약 정보를 앱 실행과 동시에 한 화면에서 즉각적으로 확인할 수 있습니다.
2. 초고속 QR 출입증
- 로컬 세션 캐싱 및 백그라운드 갱신(당겨서 새로고침) 알고리즘을 적용하여, 네트워크 지연을 최소화하고 즉각적으로 동적 QR 코드를 로드합니다.
3. 스마트 열람실 좌석 관리
- 기존에 웹뷰(WebView) 방식으로 느리게 동작하던 좌석 맵을, HTML 데이터를 직접 파싱(Scraping)하여 네이티브 UI(Native Tile View)로 렌더링함으로써 빠르고 부드러운 조작감을 구현했습니다.
- 비콘(Beacon) 인증 기반의 좌석 예약, 시간 연장, 퇴실 기능을 지원하며, 잔여 시간을 직관적인 프로그레스 바(Progress Bar) 형태의 UI로 제공합니다.
4. 도서 대출 현황 통합 관리
- 대출 중인 도서 목록과 반납 일자, 남은 기한(D-Day 뱃지)을 확인하고, 앱 내에서 원클릭으로 도서 연장 신청을 진행할 수 있습니다.

[기술적 차별점]

- 독자적인 보안 통신 이식: 학교 API 서버와의 통신에 사용되는 고유의 암호화 알고리즘(Sponge 알고리즘)을 역공학하여 앱 내부 로직으로 완벽하게 이식(TypeScript 이식)함으로써 안전하고 독립적인 통신 환경을 구축했습니다.
- 최신 상태 관리 기법 적용: React Query를 활용하여 실시간으로 변동되는 좌석 현황과 대출 데이터를 효율적으로 캐싱하고 동기화합니다.
